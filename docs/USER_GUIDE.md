# 用户指南

## 快速开始

### 1. 创建任务

点击首页底部的「立个 Flag」按钮，输入任务名称和截止日期。

### 2. 记录"鸽了"

当你没能完成任务时，点击任务卡片上的「我颓了」按钮。系统会要求你拍照作为"呈堂证供"。

### 3. 分享给死党

拍照后，系统会生成分享链接。发送给你的死党，让他来判决你的命运。

### 4. 接受判决

- **算他过了**：死党放你一马，没有惩罚
- **立刻处刑**：手机锁定 30 秒，强制反省

## 常见问题

### Q: Web 端无法使用相机？

A: 浏览器安全限制，Web 端会自动调用文件选择器选择图片。这是正常行为。

### Q: 照片会上传到哪里？

A: 照片会上传到 Supabase 云存储，只有你分享的链接才能访问。

### Q: 死党判决后多久生效？

A: App 每 2 秒检查一次判决状态，通常几秒内就会生效。

### Q: 订阅后可以退款吗？

A: 订阅通过 Apple/Google 处理，请参考各自的退款政策。

### Q: 数据会丢失吗？

A: 任务数据存储在本地设备，不会丢失。照片存储在云端，即使换设备也能通过链接访问。
